# stevedore-cli

The command-line interface for
[`stevedore`](https://github.com/ojhermann-org/stevedore) — move secrets between
password managers and vaults. Installs the `stevedore` binary.

```console
$ stevedore stores
sources: dashlane
sinks:   (none yet)
```

> Early scaffold. Moving secrets needs a source and a sink; Dashlane is modelled
> and the second store is not, so there is no route to offer yet and no
> `migrate` command that could honestly succeed.
